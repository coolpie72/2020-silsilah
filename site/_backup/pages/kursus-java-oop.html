Kursus
JAVA-201 - Pemrograman Berorientasi Objek (OOP): berbasis Java
Apa yang dipelajari?
  • Konsep class dan objek
  • Konsep atribut, method
  • Instansiasi, new operator, constructor
  • Inheritance
  • Abstract Class
  • Interface
  • Polymorphism
  • Intro Design Pattern
  • Design Pattern: Singleton
  • Design Pattern: Factory Pattern
Untuk Siapa?
  • Orang yang sudah menguasai pemrograman Java dasar, ingin beranjak ke tingkat intermediate dan advanced
  • Mahasiswa TI yang sedang belajar pemrograman, khususnya pemrograman lanjutan yang berbasis OOP
  • Mahasiswa TI pemula, untuk menguatkan sejak awal dasar pemrograman,
    sebagai bekal bimbingan kuliah, persiapan memasuki dunia kerja
Apa Manfaatnya?
  • Paradigma OOP adalah paradigma lanjutan dari paradigma pemrograman prosedural, perlu adanya perubahan pandangan yang benar untuk menguasai paradigma OOP
  • Menggunakan bahasa Java secara sesuai dengan kelebihannya yaitu mengembangkan program dengan basis objek
  • Program/kode yang dibangun dengan berbasis objek yang baik sangat mudah untuk dipahami, terstruktur dengan baik, intuitif dan mudah diubah jika diperlukan
  • Menguatkan konsep pemrograman OOP secara umum, agar peserta juga dapat mudah adaptasi ke bahasa pemrograman berbasis OOP lainnya,
    seperti C++, PHP, Scala, C#, Python, Javascript, dll.
  • Bahan belajar/contoh yang dipakai bukan hanya teori tapi juga disesuaikan dari pengalaman praktek nyata di lapangan
  • Tidak hanya belajar dasar OOP saja, penerapan OOP yaitu Design Pattern juga diikutkan. Penguasaan/pengetahun tentang Design Pattern adalah penting untuk programmer tingkat advanced
