JAVA-301 Java Platform
Apa yang dipelajari?

Java Platform mengajarkan hal-hal berikut:
  • Konsep java secara spesifik: sejarah java, Java Virtual Machine (JVM), Java Runtime Environment (JRE), Java Development Kit (JDK), java compiler (javac), java interpreter (java), automatic garbage collection
  • JVM options: tuning runtime JVM
  • Java monitoring: memory usage, jconsole, jvisualvm
  • Pengenalan development dengan java: penggunaan IDE standar industri: Eclipse, IntelliJ IDEA
  • Pengenalan deployment dengan java: bagaimana cara aplikasi java yang sudah dikembangkan, kemudian dipackaging, dan disetup di komputer sasaran. Konsep file jar, build program: ant, maven, gradle
Untuk Siapa?
  • Programmer yang sudah menguasai java secara bahasa, namun ingin menguasai java dari sisi platform secara lebih detail dan lengkap.
Apa Manfaatnya?
  • Mengetahui dan menguasai aspek-aspek di luar development: seperti packaging, monitoring, deployment, dibahas agar peserta dapat mendapatkan gambaran yang lebih utuh tentang pengembangan dan pemeliharaan aplikasi berbasis Java.
  • Peserta akan naik dari level level "user" (hanya mengenal aspek bahasa Java saja) ke level profesional, yaitu pengetahuan yang terpakai di dunia kerja nyata.
